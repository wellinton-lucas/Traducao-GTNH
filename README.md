# Tradução-GTNH  
Tradução para o português **de todas as quests** do modpack **GregTech: New Horizons**

Todas as quests até a versão **2.8.4** do modpack estão traduzidas. Essa tradução foi feita usando **IA**, tentando ao máximo manter o **mesmo tom dos textos originais**, além das **explicações técnicas**. No geral, a tradução ficou bem decente e já ajuda **pra caramba** quem não manja de inglês.

Claro que não se compara a uma tradução totalmente revisada e bonitinha. Se você quiser algo mais polido, recomendo entrar no **Discord oficial do modpack** e dar uma olhada na sessão de tradução para português. Tem uma galera dedicada lá fazendo esse trampo, porém no momento a tradução oficial ainda está em torno de **30% concluída**. Foi justamente por isso que resolvi fazer essa versão.  
Agora, se quando você estiver lendo isso a tradução oficial já estiver completa, **vale mais a pena usar a deles**.

## Instalação

É bem simples:

1. Baixe o arquivo **`pt_BR.lang`**
2. Coloque no caminho: **`config/txloader/load/betterquesting/lang/`**
(dentro da pasta onde o modpack está instalado)
3. Dentro do jogo, mude a linguagem para **Português Brasil**
4. Todas as quests devem aparecer traduzidas
5. Se divirta e sofra no modpack :)
